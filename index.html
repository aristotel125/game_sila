📖 Инструкция по использованию игры

🎮 Начало игры

Шаг 1: Выбор количества участников

При загрузке страницы вам нужно:

  • Выбрать количество участников (от 2 до 10 игроков)
  • Нажать кнопку "Далее →"

Шаг 2: Ввод имён участников

Для каждого участника необходимо указать имя:

  • Введите имя в соответствующее поле
  • Важно: Если не все имена указаны, игра попросит вас заполнить все поля
  • После того как все имена введены, нажмите "Начать игру 🎮"

🎯 Игровой процесс

Размещение фишек

  • После запуска игры все фишки с именами игроков появляются в левом верхнем углу игрового поля
  • Каждый игрок имеет свой уникальный цвет

Управление фишками

  • Ведущий может свободно перетаскивать фишки по игровому полю
  • Просто зажмите фишку мышкой и перетащите в нужное место

🔍 Масштабирование поля

Для удобства игры вы можете приближать или отдалять игровое поле:

Способ 1: Колесико мыши

  • Прокрутите колесико вверх — приблизить
  • Прокрутите колесико вниз — отдалить

Способ 2: Кнопки управления (справа)

  • + — приблизить поле
  • − — отдалить поле
  • 🔄 — вернуть к исходному масштабу (100%)

Перемещение по полю:

  • Для перемещения зажмите левую кнопку мыши и передвигайтесь по полю

⚠️ Важно знать

Перезагрузка страницы

При перезагрузке страницы все данные сбрасываются:

  • Расположение фишек
  • Количество игроков
  • Имена участников
  • Текущий масштаб

❗ Не перезагружайте страницу во время игры!

💡 Совет: Если хотите начать игру заново, просто перезагрузите страницу.

🎥 Трансляция игры

  • В процессе игры только ведущий транслирует экран с игровым полем
  • Остальные участники видят трансляцию и следят за ходом игры
  • Ведущий управляет фишками всех игроков

👥 Формат игры

Игра предназначена для использования только с ведущим:

  • Один человек (ведущий) управляет полем
  • Ведущий транслирует экран остальным участникам
  • Все игроки следят за процессом через трансляцию
